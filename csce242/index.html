Assignment 8 - Arrays

Help save the animals! This webpage demonstrates the use of arrays by populating a list of animal images, before and after they've been treated by the vet. Click on an image to see the transition!

Projects

Project 1 - Topic Selection

The topic for this semester's project is designing a website for a pizzeria. The goal is to make a website where customers can view the menu, place orders, and staff can organize data.

Project 2 - Wireframes

MOCKITT wireframe draft of the pizzeria website. The wireframe demonstrates the possible appearance of the website, including the pages, options, and ideas.

Project 3 - HTML and CSS

Implementing the wireframe into an actual website.

Project 4 - Colors, Pictures, Text

Focusing more on the appearance of the website. Updating with actual images and text, as well as adding a colorscheme.

Project 5 - Improve Your Website

Fixing certain elements on the site based on peer feedback.
